- **`jac ninja` easy mode: VSCode look pack**: `jac ninja --easy` now looks the part as well as typing the part -- a VSCode Dark+ colorscheme (blue status bar included) swapped live against the stock look by `:NinjaEasy on/off`, LSP breadcrumbs in the winbar with an EASY badge, an auto-opening file-tree sidebar (`ctrl+b` / `ctrl+shift+e`), a persistent bottom terminal panel (`` ctrl+` `` / `ctrl+j`), and editor splits (`ctrl+\`) -- the full VSCode geometry.
- **LSP: stdio server no longer responds to responses**: replies to server-initiated requests (such as `workspace/semanticTokens/refresh`) were echoed back as orphan `{id, result: null}` frames, a JSON-RPC violation that clients surfaced as errors (`NO_RESULT_CALLBACK_FOUND` in Neovim, where the multi-line echo blocked the TUI at a hit-enter prompt). Messages without a `method` are now consumed as the responses they are.
