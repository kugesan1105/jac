- **Fix: bound `jac-mcp` GitHub example cache and defer guide-store load**: the module-global `_github_cache` that grew without bound in deployed installs fetching examples from GitHub is replaced by a bounded `functools.lru_cache(maxsize=64)` on `_fetch_example_github`. `ResourceProvider` indexing is now lazy and idempotent (`_ensure_indexed` / `_indexed`), so the jaclang guide store is imported only on first resource/doc access instead of at server startup, and the doc tools share a single `get_resource_provider()` instance.
